Tutoriel : Phase d'entrée du jeu

1. Saisie du pseudo Lorsque vous cliquez sur le bouton "Start", une fenêtre apparaîtra vous demandant d'entrer votre pseudo. C'est le nom sous lequel vous serez connu dans le jeu. Tapez votre pseudo dans le champ prévu à cet effet, puis appuyez sur le bouton "Valider" pour continuer.

2. Choix de la classe Après avoir entré votre pseudo, vous serez invité à choisir la classe de votre personnage. Vous avez le choix entre plusieurs classes, chacune avec ses propres compétences et caractéristiques. Sélectionnez la classe qui correspond le mieux à votre style de jeu, puis appuyez sur le bouton "Valider".

3. Mise en état "Ready" Une fois que vous avez choisi votre classe, vous serez prêt à commencer l'aventure ! Appuyez sur le bouton "Ready" pour indiquer que vous êtes prêt à partir à l'aventure et à affronter les défis qui vous attendent
